Classical Text Ciphers: Caesar / XOR / Rail Fence

  • Caesar Cipher: Each letter in the plaintext is shifted a fixed number of positions in the alphabet.
  • XOR Cipher: Applies bitwise XOR between each character of plaintext and key; same function used for encryption and decryption.
  • Rail Fence Cipher: A transposition cipher that writes characters in a zigzag pattern across multiple rows (rails).

Credit: Summary adapted from lecture notes of DES332: Computer and Network Security, taught by Dr. Nguyen Duy Hung, SIIT.
